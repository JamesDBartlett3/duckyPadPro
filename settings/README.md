# Settings

This directory contains various configuration and settings files for the duckyPad Pro device.

## Contents

- **Device settings**: Global device configuration files
- **Customization**: Color schemes, themes, and display settings
- **Backup configurations**: Template settings files

## File Formats

Settings files typically use `.txt` or `.ini` format, depending on the specific configuration type.
