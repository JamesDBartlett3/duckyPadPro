# Scripts

This directory contains standalone duckyScript files for various tasks and automations.

## Organization

Scripts are organized by category:

- **productivity/**: Scripts to enhance productivity
- **development/**: Scripts for developers
- **system/**: System administration scripts
- **media/**: Media control scripts

## File Naming

Use descriptive names that clearly indicate what the script does:
- `open-terminal.txt`
- `launch-browser.txt`
- `volume-control.txt`

## Documentation

Each script should include comments at the top explaining:
- What the script does
- Any prerequisites or requirements
- Author and date (optional)

Example:
```
REM Script: Open Terminal
REM Description: Opens a new terminal window
REM Author: Your Name
REM Date: 2025-01-01

COMMAND SPACE
DELAY 500
STRING terminal
DELAY 200
ENTER
```
