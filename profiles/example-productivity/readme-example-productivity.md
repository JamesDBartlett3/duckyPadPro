# Example Productivity Profile

This is an example profile demonstrating how to organize and structure a duckyPad Pro profile.

## Keys

- **Key 1**: Open Terminal
- **Key 2**: Open Browser
- **Key 3**: Take Screenshot
- **Key 4**: Lock Screen

## Usage

1. Copy this profile directory to your duckyPad Pro device
2. Select the profile from the device menu
3. Press the configured keys to execute the scripts

## Customization

You can customize the scripts by editing the individual key files (`key1.txt`, `key2.txt`, etc.).

## Notes

- Scripts are written for macOS. Adjust key combinations for Windows or Linux as needed.
- Add delays between commands if actions aren't executing reliably.
